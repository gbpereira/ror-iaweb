### Operadores

```ruby
# operadores de atribuição
x = 6           # atribuição regular
x += 6          # atribui a adição
x -= 12         # atribui a subtração
x *= 4          # atribui a multiplicação
x /= 8          # atribui a divisão
x **= 2         # atribui a exponenciação
x %= 3          # atribui o módulo
```
