### Variáveis

```ruby
# tipos de variáveis
year = 2015     # variável local
_year = 2015    # variável local
@year = 2015    # variável de instância
@@year = 2015   # variável de classe
$year = 2015    # variável global

YEAR = 2015     # constante
Year = 2015     # constante
```

```ruby
> defined? var # => "local-variable"
```
