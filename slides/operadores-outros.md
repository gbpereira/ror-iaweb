### Operadores

```ruby
# comparadores de regex
/c/ =~ 'abcde'  # retorna 2
/c/ !~ 'abcde'  # retorna false

# operadores lógicos
&&              # E lógico
||              # OU lógico
!               # Negação lógica

# operadores de range
1..10           # range inclusivo
1...10          # range exclusivo
```
