### Operadores

```ruby
# operadores aritméticos
7 + 5           # adição
20 - 8          # subtração
2 * 6           # multiplicação
12**2           # exponenciação
12 % 5          # módulo

# divisão com truncamento
25 / 2          # => 12

# divisão sem truncamento
25.0 / 2        # => 12.5
25 / 2.0        # => 12.5
```
