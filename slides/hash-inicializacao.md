### Hash

```ruby
# inicializando hash
hash = {}                 # inicializa um hash vazio
hash = Hash.new           # inicializa um hash vazio
hash = {one: 1, two: 2}   # inicializa um hash com valores

# podem conter tipos diferentes de dados
hash = { int: 1, range: (1..4), arr: [1, 2, 3], h: {a: 'a'} }
```
